# Многомерни масиви
## Двумерни масиви
```c++
int matrix1[3][4]; // създаме матрица с 3 реда и 4 колони

int matrix2[3][4] = { { 1, 2, 3, 4 }, { 9, 8, 7, 6 }, { 11, 12, 13, 14 } }; // изреждаме редовете

int matrix3[3][4] = { 1, 2, 3, 4, 9, 8, 7, 6, 11, 12, 13, 14 }; // изреждаме елементите

int matrix4[][4] = { 1, 2, 3, 4, 9, 8, 7, 6, 11, 12, 13, 14 }; // най-лявата спецификация на дължина може да се изпусне
```

## N-мерни масиви
```c++
int cube[4][4][4];
```

## Достъп до елементите на многомерен масив
Тук също имаме константен достъп до елементите.
```c++
arr[1]; //едномерен масив
arr[1][3]; // двумерен масив
arr[1][3][4]; // тримерен масив
```
