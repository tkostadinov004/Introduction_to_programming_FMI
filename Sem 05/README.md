# Масиви
- наредена последователност от елементи с еднакъв тип

Деклариране и инициализация: 
```c++
int arr[5]; // създава се масив от 5 елемента, всеки от които е цяло число (int)
int arr[5] = {2, 4, 6, 8, 10};
int arr[] = {2, 4, 6, 8, 10}; // размерът може и да се пропусне

int arr[]; // грешка
int arr[5] = {2, 4, 6, 8, 10, 12}; // отново грешка
```

### Относно размера на масива
Задължително е размера на масива да е константа, чиято стойност се знае преди компилацията на програмата. Тоест, следното парче код е вярно: 
```c++
int arr[5] = {2, 4, 6, 8, 10};

constexpr unsigned SIZE = 5; // стойността е ясна ПРЕДИ компилацията
int anotherArr[SIZE] = {1, 2, 3, 4, 5};
```

А следното парче код ще е грешно:
```c++
unsigned size;
cin >> size;

int arr[size];
```
### Достъп до елементите на масива
В масивите имаме константен достъп до всеки елемент на масива. Достъпът става чрез индекси.
```c++
int arr[] = {1, 2, 3, 4, 5};
arr[1] = 100; // {1, 100, 3, 4, 5}
cout << arr[1]; // принтира се елементът на 1-ви индекс (100)
```

***⚠️ arr е указател към първия елемент***
```c++
int arr[] = {1, 2};
cout << arr; // няма да принтира масива или първия елемент, а адреса в паметта на първия елемент
```

Достъпът до елемент става чрез смятане на адреса му: </br>
Адресът на arr[i] е ```arr + i * sizeof(типът на масива)```

### Подаване на масиви във функции
```c++
void print(const int arr[], unsigned size)
{
    for (unsigned i = 0; i < size; i++)
    {
        cout << arr[i] << " ";
    }
}

void power(int arr[], unsigned size)
{
    for (unsigned i = 0; i < size; i++)
    {
        arr[i] *= 2;
    }
}

int main()
{
    constexpr unsigned SIZE = 5;
    int arr[SIZE] = {1, 2, 3, 4, 5};

    print(arr, SIZE); // 1 2 3 4 5
    power(arr, SIZE);
    print(arr, SIZE); // 2 4 6 8 10
}
```
***⚠️ Масивите се подават по адрес във функциите, тоест всяка промяна върху елементите на масива ще окаже влияние и извън функцията, в която са направени промените*** </br>
***⚠️ Когато се подават масиви във функции е добре да се подават с ключовата дума const, ако няма да се променят елементите на масива (както е във функцията print)*** </br>
