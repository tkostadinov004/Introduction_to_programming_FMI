# Функции
Когато пишем дадена програма е доста удобно да преизползваме код, който вече сме написали, за да не се налага да го пишем повторно на много места. </br>
В С++ съществуват такива средства за преизползване на код и те се наричат **функции**.

### Какво е функция?
Функциите са:
* именувани групи от инструкции, които изпълняват дадена функционалност.
* код, който е за общо предназначение

Синтаксис: 
```c++
<тип на връщане> <име>(<параметър1>, <параметър2>, ..., <параметърN>)
{
    // тяло
}
```
***Всеки параметър се състои от тип и име (идентификатор).*** </br></br>
Пример за функция:
```c++
int min(int a, int b)
{
    return a <= b ? a : b;
}
```

### Подаване на параметри на функция
Параметрите на функция могат да бъдат подавани по два начина - по копие и по референция. </br>
#### Подаване по копие
Когато се подават параметри на функция по копие, всякакви промени върху тях не се отразяват извън функцията.</br>
Пример:
```c++
void f(int a)
{
    a--;
}

int main()
{
    int a = 10;
    f(a);
    cout << a;
}
```
⚠️ ***Тук програмата ще принтира "10", а не "9", понеже при подаване на параметъра на функцията му се прави копие, което се намира на друго място в паметта и не е свързано по никакъв начин с променливата отвън, освен по това, че имат равна стойност.*** </br>
Ако искаме обаче да напишем функция, в която промените по някои от параметрите ѝ да се отразява и извън нея, трябва да разберем какво е *подаване по референция*.
#### Референция и подаване по референция
##### Какво е референция?
Референцията (препратка, псевдоним) е друго име за променлива.
```c++
int main()
{
    int a = 10;
    int& b = a;
    b--;
    cout << a;
}
```
Всяка промяна на B се отразява на А, понеже В сочи към А.
##### Подаване по референция
Можем да подаваме параметри на функция по референция:
Пример:
```c++
void f(int& num)
{
    num--;
}

int main()
{
    int a = 10;
    f(a);
    cout << a;
}
```
⚠️ ***Тук програмата ще принтира "9" вместо "10", понеже параметърът num е подаден по референция и всяка промяна върху него ще се отрази и върху променливата, към която сочи той*** </br>

**2.** *Двоича експоненциация* е трик, често използван в състезателното програмиране, който ни позволява да изчисляваме по-бързо степени от вида $`a^n`$. Наивният подход за повдигане на степен представлява умножение на числото $`a`$   $`n`$ пъти - $`a^n = a \cdot a \cdot a ... a`$. Този подход обаче не е подходящ за много големи числа.</br>Нека напишем числото $`n`$ в двоична бройна система. 
Например,</br>$`3^{13} = 3^{1101_2} = 3^{1000_2 + 0100_2 + 0001_2} = 3^{1000_2} \cdot 3^{0100_2} \cdot 3^{0001_2} =3^8 \cdot 3^4 \cdot 3^1`$. 
Понеже $`n`$ има $`\lfloor \log_2 n \rfloor + 1`$ на брой цифри, ще трябва да извършим $`log_2(n)`$ пресмятания, вместо $`n`$. Всяко число в двоична бройна система може да се образува като сбор от степени на двойката. 
</br> Например, числото $`13 ={1101_2}`$ може да се запише като $`{1101_2} = {1000_2} + {0100_2} + {0001_2} = 8 + 4 + 1`$. </br></br> Изчислението е дадено по следния начин:</br></br>
$`a^n = \begin{cases}
1 &\text{, ако } n = 0 \\
\left(a^2\right)^{\frac{n}{2}} &\text{, ако } n > 0 \text{ и } n \text{ е четно}\\
a \cdot \left(a^{n-1}\right) &\text{, ако } n > 0 \text{ и } n \text{ е нечетно}\\
\end{cases}`$</br></br>
Напишете функция, която изчислява степента $`a^n`$, където $`a`$ е естествено число и $`n`$ е цяло число.</br></br>
